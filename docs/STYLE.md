# Scholar Speak — Style Reference (BEHAVE Candy)
> ネオン・キャンディ・ポップ。コバルトブルー1色が全ての文字・枠・アイコンを担い、ライムは「動く/叫ぶ」瞬間だけのアクセント。背景は白ではなく淡いラベンダーで、内側から光っているように見せる。Theme: light固定。

## Colors
--cobalt: #061fff;        /* 全ての本文・見出し・枠線・アイコン・塗りCTA。ブランドの唯一の声。黒・グレーの文字は使わない */
--lime: #d3ff56;          /* 塗りアクション・選択中状態・注目要素だけ。面(背景)には使わない=句読点として使う */
--blush: #ffcfd1;         /* カード塗り(単語カード等)。コバルト/ライムの強さを和らげるキャンディピンク */
--lavender: #efe5ff;      /* ページ全面の背景。白やグレーの代わり。全セクションがこの上に乗る */
--lilac-bloom: #ceb3ff;   /* 一段持ち上げたカード面・ホバー(影を使わず面で高さを出す) */
--soft-lilac: #dfceff;    /* 入力欄の背景 */
--paper: #ffffff;         /* 塗りコバルトCTA内の文字・写真上の見出しのみ。ページ背景には使わない */

## Typography
- 単一書体: Inter / 'DM Sans' / 'Space Grotesk' 系。font-family: 'DM Sans', Inter, system-ui, sans-serif;
- ウェイト: 本文400 / ナビ・カード見出し500 / ディスプレイ700
- **署名 = 大見出しの負のトラッキング**: 56px以上で letter-spacing: -0.05em(モバイルの単語表示も同様に詰める)。32px=-0.022em, 24px=-0.02em, 16px以下=-0.015em。大きい見出しほど詰める(開かない)
- スケール: caption12 / body16(lh1.5) / subheading24 / heading-sm32 / heading56 / display86。モバイル単語表示は clamp(40px, 15vw, 72px) を700で -0.05em

## Shapes & Layout
- radius: カード20px / ボタン・入力6.95px / アロー(丸)25px
- **影は一切使わない**。深さは①ラベンダー3段(#efe5ff→#dfceff→#ceb3ff)②カードの1pxコバルト枠 で表現
- card padding 29px / element gap 20–30px / section gap 大きめ(可読重視で調整可)
- 全てのカードに1pxコバルト枠+20px角
- 塗りライムを広い面の背景に使わない。第3のアクセント色を足さない(コバルトとライムだけ)

## Components mapping (Scholar Speak)
- ページ背景: Lavender Mist(#efe5ff)全面。ライト固定
- アプリバー: ラベンダー地、中央ワードマーク「SCHOLAR SPEAK」コバルト700・詰めトラッキング。枠・影なし
- 単語カード表: Blush Puck(#ffcfd1)地+1pxコバルト枠20px角。単語=コバルト700の超特大(-0.05em、fitTermFontで途中折らない)、IPA=コバルト。🔊はコバルトのアウトライン丸(radius25px)
- 単語カード裏/辞書カード: 背景ラベンダー、カードはBlush or Soft Lilac+1pxコバルト枠。本文は全てコバルト文字。セクション見出しラベルはコバルト500小。context_quote(使用例)は塗りコバルト面+白文字の帯(唯一の白文字使用箇所)
- クイズ選択肢: Soft Lilac(#dfceff)地+1pxコバルト枠+コバルト文字。ホバー/選択でLilac Bloom。正解=Sour Lime塗り+コバルト文字、不正解=Blush強め+コバルト文字
- 主要アクション(次へ/学習開始/覚えた): **塗りライム(#d3ff56)+コバルト文字**、6.95px角(ピル禁止)。同一画面で2つ目のCTAが要る時はコバルト塗り+白文字と組み合わせ、同色2つを隣接させない
- 「もっと詳しく」等の副アクション: コバルトのアウトライン(透明地+1pxコバルト枠+コバルト文字)
- モードトグル(4択/2択/カード): コバルト枠のセグメント、選択中セグメントはライム塗り+コバルト文字
- デッキ一覧: 各デッキ=Blush Puckカード+1pxコバルト枠20px角。デッキ名コバルト500、件数コバルト400小。バッジ無し
- 進捗バー: ラベンダー地の溝+ライム塗りの進捗(影・グラデ不可)
- 下部タブナビ: ラベンダー地、コバルトのアイコン/ラベル(500小)、選択中はライム下線 or ライム塗りチップ+コバルト文字。枠・影なし
- 入力(検索など): Soft Lilac地・枠なし・6.95px角、プレースホルダはコバルト

## Don't
- 黒・グレーの文字を使わない(全てコバルト)/影を使わない(1pxコバルト枠で深さ)/ライムを広い背景面に使わない/大見出しでトラッキングを開かない/9999pxのピルCTAを作らない(6.95px)/第3のアクセント色を足さない/白をページ背景にしない
